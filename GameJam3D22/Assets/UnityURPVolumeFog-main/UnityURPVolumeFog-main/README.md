# Unity URP Volume Fog - Modified Version

This repository is a modified version based on the original project:

Original project: https://github.com/CristianQiu/Unity-URP-Volumetric-Light

Original author: CristianQiu

The original project provides a Unity URP volumetric lighting / fog package.  
This version contains my own modifications for use in my Unity projects.

## Modifications

- Adjusted the package for my own URP project workflow.
- Modified shader / rendering code for my project requirements.
- Packaged it as a reusable Unity package for easier use across different projects.

## Installation

In Unity, open:

Window > Package Manager > + > Install package from git URL

Then use:

https://github.com/Skyshin1/UnityURPVolumeFog.git
